Week 2: JavaScript Fundamentals & Backend Development

This folder contains JavaScript programs that help in learning intermediate and advanced concepts of programming. Each file focuses on a specific task.

shoppingCart.js : Filters products that are in stock, calculates total price of all items in the cart using map() and reduce().

studentArray.js : Filters students who passed (marks ≥ 40), adds grade field (A, B, C, D) based on marks, and calculates average marks.

marksOps.js : Filters passing marks (≥ 40), adds 5 grace marks to each student using map(), and finds the highest mark using reduce().

courseOps.js : Filters courses with name length greater than 5 characters, converts all course names to uppercase, and joins them into a single string.

employeePayroll.js : Manages employee salary data, calculates bonuses, and processes payroll using array methods.

movieStreaming.js : Filters movies by genre or rating, transforms movie data, and finds specific movies in the database.

transactionAnalyzer.js : Analyzes bank transactions, filters by amount or type, calculates total deposits and withdrawals.

bookClassDemo.js : Creates a Book class with properties (title, author, pages, isAvailable) and methods (borrow, returnBook, getInfo, isLongBook). Manages a library collection.

restparameter.js : Shows how to use the spread operator (...) to copy arrays and objects, and the rest parameter to handle any number of function arguments.

shallowDeepClone.js : Demonstrates the difference between shallow copy (using spread operator) and deep copy (using structuredClone()) with nested objects.

setTimeout.js : Simulates an exam portal where "Exam submitted successfully" appears immediately, then "Evaluating answers..." after 2 seconds, and "Result: Pass" after 4 seconds.

otpCountdown.js : Simulates OTP sending with a 10-second countdown timer using setInterval(), allowing resend only after the countdown ends.

todo application

app.js : The main entry point for the backend server. Sets up Express, connects routes, and starts the server on a specific port.

ProductAPI.js : Creates a complete REST API for managing products with all CRUD operations (GET, POST, PUT, DELETE).

task.js : Defines data models/schemas for tasks or products used in the application.

validator.js : Contains validation functions and middleware to check incoming data before processing.
