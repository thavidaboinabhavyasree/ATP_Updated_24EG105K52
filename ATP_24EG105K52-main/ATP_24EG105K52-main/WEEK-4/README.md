Week 4: HTML Practice

This folder contains HTML programs that help in learning the basics of web page structuring. Each file focuses on a specific task.

shoppingList.html : Demonstrates nested unordered lists to show hierarchical relationships between items. Groups food items into categories like Fruits (Apple, Banana, Mango, Orange) and Vegetables (Spinach, Cauliflower, Beetroot), along with other items like Cereals and Nuts. The page title is "Grocery List".

myFirstWebpage.html : Creates a simple webpage demonstrating basic HTML tags including paragraphs, headings, bold text, hyperlinks to Google, and unordered lists of fruits. Shows how to structure content with text formatting and lists. The page title is "My First Webpage".

pochiTheCat.html : Creates a profile webpage for "Pochi The Cat" featuring an image aligned to the right, introduction section, profile list with favorite food (smoked salmon), hobbies (watching fishing on ESPN, snacking on garden flowers, monitoring the apartment parking lot), hidden talent (Karaoke), and external links to animal shelter organizations (Seattle Animal Control Shelter and Humane Society of the United States). Includes proper page structure with headings, ordered lists, and horizontal rule. The page title is "Pochi The Cat"

Concepts Learned

HTML Document Structure : Using DOCTYPE declaration, html, head, and body tags to create a proper webpage structure.

Headings : Using h1 and h2 tags to create hierarchical headings and organize content.

Paragraphs : Using p tags to write descriptive text content with proper spacing.

Text Formatting : Using bold tags (b) to emphasize important words and italic tags (i) for styling text.

Hyperlinks : Creating anchor tags (a) with href attributes to link to external websites and resources.

Images : Embedding images using img tag with src attribute and alt text for accessibility, including floating images with style attributes.

Lists : Creating unordered lists (ul) for bullet points and ordered lists (ol) for numbered sequences.

Nested Lists : Creating multi-level lists by placing unordered lists inside list items to show hierarchical data.

Horizontal Rule : Using hr tag to create a visual separator between sections.

Page Metadata : Setting title, viewport, and character encoding for responsive and properly rendered webpages.
